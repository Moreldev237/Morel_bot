
# MorelBot 🤖

Un chatbot intelligent basé sur GPT-4.1 via OpenRouter, avec une interface conversationnelle personnalisée.

## ✨ Fonctionnalités

- Interface conversationnelle élégante avec votre photo en arrière-plan
- Intégration avec l'API OpenRouter (GPT-4.1)
- Backend Django robuste
- Frontend moderne avec TailwindCSS
- Historique des conversations
- Design responsive

## 🚀 Installation

### Prérequis
- Python 3.10+
- pip
- Clé API OpenRouter
